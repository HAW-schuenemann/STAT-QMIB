# QMIB Exercises

This folder contains solution files for the QMIB exercises.

The exercises are organised in subfolders according to the PDF document they belong to. This structure is used to keep the task numbering consistent with the original exercise documents.

For example, if an exercise is labelled as `Exercise 5` in a specific PDF, the corresponding solution file can be found in the folder for that PDF.

## Folder structure

```text
QMIB/
├── PDF_1/
│   ├── exercise_01.R
│   ├── exercise_02.R
│   └── ...
├── PDF_2/
│   ├── exercise_01.R
│   ├── exercise_02.R
│   └── ...
└── README.md
```

## Note

Exercise numbers may repeat across folders because each PDF document has its own numbering. The folder structure makes clear which original document each solution refers to.

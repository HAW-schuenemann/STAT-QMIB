## 3. R Help and Documentation

### Built-in R help (inside RStudio)
Use `?` followed by the function name:

```r
?mean
?quantile
?freq
```

This opens the official documentation directly in RStudio.

### Online documentation
https://rdocumentation.org/

Helpful if you want to search for functions without opening RStudio.

## 5. Community Support and Error Messages

### Stack Overflow (R questions)
https://stackoverflow.com/questions/tagged/r

Helpful when encountering error messages.

Recommended search strategy:
- Copy the error message
- Add the function name
- Add "R"

Important:
Do not copy solutions blindly.
Use them to understand the problem and adapt the solution
to the methods used in this course.

## 6. How to Use External Resources Effectively

External resources are meant to support learning, not replace it.

Recommended approach:
1. Try to solve the task on your own first.
2. Compare your solution with the course repository.
3. Use external resources only if something is unclear.
4. Rewrite the solution in your own words and code.

Active practice is essential.
Reading code alone is not sufficient to learn R.

---

## 7. Reminder for Exam Preparation

- Practise writing code yourself
- Do not memorise solutions line by line
- Focus on understanding:
  - which function to use
  - why it is used
  - how to interpret the output

The repository and these resources are tools to support you,
but they do not replace independent practice.
